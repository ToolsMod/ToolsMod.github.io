Esp

Setting Description
colorPlayers The color of other players.
colorEntitys The color of entitys.
colorItems The color of items.
players If players should be rendered.
entitys If entitys should be rendered.
items If items should be rendered.

How to use

Renders Entitiys/Players and/or items with an outline through blocks.

The settings for the color and which types should be rendered, can be changed.
