Headwriter

Supports 34 different Styles

Requirements Description
Creative The gamemode must be creative
Build rights The user must be able to build in the world

How to use

Go into the gui of the headwriter (Right click on the Button) in the section Special

There you can see 2 Tabs

On the Text-tab you can define the text you would like to write.

On the Type-tab you can select the style to write your Text with.

When you have choosen the style and your text, click on the Start button.

The headwriter will give you the first head.

Now you can place the heads and the headwriter will automatically provide you with the next head.

When the Text is finished, the headwriter will deactivate itself.
