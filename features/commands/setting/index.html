Setting

#Setting <module> [setting] [value]

Keyword Description
<module> The module of which you want to see or change the settings.
[setting] The setting of which you want to know the value or want to change.
[value] The new value of the setting.

Some modules have settings that can't be changed otherwise yet.

This command is there to view and edit those.
