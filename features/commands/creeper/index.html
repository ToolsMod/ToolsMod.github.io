Creeper

#Creeper <x> <y> <z> [radius] [fuse]

Keyword Description
<x> <y> <z> The coordinates where the creeper should spawn.
[radius] Define the radius of the explosion.
[fuse] Define how long it will take until the creeper blows up.

Spawn a creeper that blows up immediately.

Requirements Description
Creative The gamemode must be creative
Build rights The user must be able to build in the world
