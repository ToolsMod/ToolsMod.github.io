ASSave

#ASSave <x1> <y1> <z1> <x2> <y2> <z2> <name>

Keyword Description
<x1> <y1> <z1> The first position.
<x2> <y2> <z2> The second position.
<name> The project-file name.

Saves all armor stands in the region to a project-file.

Can be loaded using the ASEdit module.
