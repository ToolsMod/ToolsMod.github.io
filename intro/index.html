Modules

The main part of the mod consists of splitted features called Modules.

This Modules can be activated by the gui.

You can open the gui by pressing the key X

A detailed description of every Module can be found here.

Some modules have settings. This settings can be modifyed by the settings-gui of every module.

The setting gui can be accessed by using a right click on the module button in the gui.

Some modules don't have settings guis but settings. This settings can be viewed and changed using the settings command.

Command

Another important part of the mod are the Commands.

The commands are typed into the chat windows and can be accessed using the prefix #.

A list of all Commands can be found here.

Also there is a help command that can be accessed by typing #Help
